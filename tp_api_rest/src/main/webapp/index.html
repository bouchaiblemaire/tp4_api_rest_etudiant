API REST


URL de l'API REST: IP_SERVEUR:port/tp_api_rest/api/employees
  • Liste tous les employées(JSON)
    • Méthode : GET, chemin:/
    • Réponse : Format JSON
  • Information de l'employée d'identifiant id
    • Méthode :GET, chemin:/id
    • Réponse : Format JSON
  • Création d'un employée
    • Méthode :POST,chemin:/
    • Information au format JSON de l'employée à créer avec le même format que ceux renvoyé par le serveur.
    • Réponse : Format JSON
  • Mise à jour d'un employée
    • Méthode :PUT,chemin:/
    • Information au format JSON de l'employée à mettre à jour, même format que ceux renvoyé par le serveur complété avec l'id de l'employée à mettre à jour.
    • Réponse : Format JSON
  • Suppression d'un employée
    • Méthode :DELETE,chemin:/id
    • Réponse : boolean
